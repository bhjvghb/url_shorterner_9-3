# 2025-07-02

## UI Redesign Prototype
- 分析了 url_shortener 项目现有 UI（Bootstrap 5 + 紫蓝渐变，典型的 AI 审美风格）
- 基于 Impeccable 设计原则创建了全新 UI 原型 `ui-prototype.html`
- 设计方向：**Warm Editorial**（温暖编辑风）
  - 暖米色背景 (#faf7f2)，赤陶色 (#d9704a) 作为强调色
  - 系统字体栈，大号自信的排版
  - 4pt 间距系统，使用 border 而非 card 进行视觉分隔
  - 不使用 Bootstrap CDN，完全自定义 CSS
- 原型涵盖：登录、注册、忘记密码、首页缩短表单、成功页、仪表盘、统计页
- 页面间可交互切换，支持复制到剪贴板、toggle 展开等交互
- **新增**：主页 Hero 区域添加港大 ECE 校徽项目归属徽章（胶囊卡片样式），图片已复制到 `static/hku-ece-logo.png`

## Template Migration — Warm Editorial Design
- 创建了 `static/style.css` 完整设计系统（~600 行），包含设计 Token、组件库、工具类
- 移除所有 Bootstrap CDN 依赖，12 个模板全部迁移至新设计
- 更新模板：index, login, register, forgot_password, reset_password, error, protected, success, dashboard, admin_dashboard, link_stats, _lang_switcher
- 保留功能：i18n、Chart.js 统计图表（改用赤陶色系配色）、仪表盘模态框（自定义实现替代 Bootstrap Modal）
- 新增组件：`modal-overlay` + `modal` 自定义模态框系统，`tabs` 标签切换组件
